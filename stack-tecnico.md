# Stack Tecnológico

## Frontend
- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- shadcn/ui
- TanStack Query
- React Hook Form + Zod
- Recharts
- Framer Motion
- PWA

## Backend
- NestJS
- TypeScript
- Prisma ORM
- PostgreSQL
- Redis (fase futura)

## Infraestructura y DevOps
- Docker
- Docker Compose
- Nginx
- GitHub Actions
- VPS o Google Cloud

## Herramientas de desarrollo
- ESLint
- Prettier
- Husky
- Commitlint
- Swagger/OpenAPI

## Justificación del stack
- Next.js y React permiten una interfaz moderna y escalable.
- NestJS ofrece una arquitectura robusta para el backend.
- Prisma simplifica la interacción con la base de datos.
- PostgreSQL es una opción sólida para datos transaccionales.
- Docker facilita el entorno de desarrollo y despliegue.

## Entorno recomendado
- Node.js 20+
- npm o pnpm
- Docker Desktop
- PostgreSQL local o servicio gestionado
